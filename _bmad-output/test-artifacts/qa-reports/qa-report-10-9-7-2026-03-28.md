## Issues Found

### Issue 1 (P2): Missing entry in `allGatewayMethods` array

**File:** `src/common/testing/event-wiring-audit.spec.ts`
**Lines:** 502-532

**Description:** `broadcastExternalPairIngestionCompleted` is registered in the `gatewayHandlers` array (line 664) for event wiring verification, but is NOT listed in the `allGatewayMethods` array (lines 502-532). The `allGatewayMethods` array is used in `beforeEach` to pre-spy all gateway handler methods with no-op implementations (via `buildAndInit`), preventing dependency errors when handlers are invoked during event wiring tests.

**Impact:** Low. The test still passes because the real handler's `this.broadcast()` iterates an empty `clients` set (no WebSocket connections in test), making it a no-op. However, this is an inconsistency — all other gateway handlers are listed in both arrays.

**Fix:** Add `'broadcastExternalPairIngestionCompleted'` to the `allGatewayMethods` array after line 531:
```typescript
// Incremental freshness (Story 10-9-6)
'broadcastFreshnessUpdate',
'broadcastStalenessWarning',
// External pair ingestion (Story 10-9-7)
'broadcastExternalPairIngestionCompleted',
```
